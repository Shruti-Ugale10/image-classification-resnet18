# image-classification-resnet18
This project demonstrates image classification using a pretrained ResNet-18 model.

## Technologies Used
- Python
- PyTorch
- Deep Learning

## Description
The model takes an input image and predicts the class using a pretrained ResNet-18 neural network.

## How to Run
1. Install required libraries
2. Run the Python script
3. Provide an input image
